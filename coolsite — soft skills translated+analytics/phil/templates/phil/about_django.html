Разработка веб-проектов с использованием фреймворка Django (Python)

Практика:
    Прошел обучение Selfedu, создал учебный проект "Women" (ссылка на главную страницу учебного проекта).
    В проекте:
  • подбор статей с фотографией;
  • статьи разделены на категории, можно выбрать отображение статей по определенным категориям;
  • создание регистрации и авторизации пользователей;
  • добавление статей на сайт с фотографией. Возможно сделать только зарегистрированным и авторизованным пользователям;
  • для маршрутизации к статьям используются слаги (slug);
  • использование MIXINs для того, чтобы убрать дублирование кода;
  • использование постраничной навигации (пагинация);
    Дополнительно:
  • данный сайт полностью сделан на фреймворке Django: здесь создаю и выкладываю проекты (в том числе и учебные). Исходный код данного проекта можно посмотреть на GitHub;
  • создал страницу с добавлением заметок (текстов) зарегистрированным и авторизованным пользователям. Можно добавлять изображение;
  • создаю приложение, в котором можно будет сохранять и смотреть динамику своих замеров тела;
  • создаю приложение со списком прочитанных книг, в котором можно будет писать свое описание книги и добавлять цитаты из книги;
  • владею HTML;
  • владею CSS;
  • владею Git & GitHub;
  • владею MySQL, SQLite, PostgreSQL;
  • фреймворк Django;
  • библиотеки NumPy, Pandas, Tkinter.
